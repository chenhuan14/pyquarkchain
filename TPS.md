firstly, we run generate_start_script.sh with the configure file to generate start_up_qc_*.sh to each node.
seconde, we run start_cluster.sh to call start_up_qc_*.sh to start cluster.
last, we can run stop_cluster.sh to stop cluster.

the directory config-file contains the configure file.
